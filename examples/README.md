# QUEASARS Examples

This folder contains a few notebooks demonstrating QUEASARS implementation of
the E-VQE algorithm using the Qiskit API.

To run them, either use the provided poetry environemnt for QUEASARS development,
or, set up a new environment using the provided `requirements.xt` which contains
only the relevant dependencies.

QUEASARS currently requires Python 3.9 until Python 3.11.

The notebook `evqe_jssp_small_examples.ipynb` contains 4- and 5-qubit examples
quickly test QC hardware using a Qiskit backend.
